# jquery-blend

Blending modes for jquery

### Examples

    // Overlay with a solid color
    $('img').blend({mode: 'overlay', input: 'rgb(114, 25, 219)'});

### Modes
    
- normal

- darken
- multiply
- colorburn
- linearburn

- lighten
- screen
- colordodge
- lineardodge

- overlay
- softlight
- hardlight
